# Animated Emoji Map

Uses `@remotion/animated-emoji` — Google Fonts Animated Emoji rendered as Lottie videos.

## Mapping by Caption Style

### Hook (attention/shock)
| Emoji | Name | Use for |
|-------|------|---------|
| 🤯 | `mind-blown` | Surprising fact, shock value |
| 😱 | `screaming` | Fear/shock opener |
| 😴 | `sleep` | "You're sleeping on this" |
| 💤 | `sleepy` | Boredom before transformation |
| 🔥 | `fire` | Hot take, trending |

### Pain (negative emotion)
| Emoji | Name | Use for |
|-------|------|---------|
| 😩 | `weary` | Frustration, exhaustion |
| 😫 | `distraught` | Deep frustration |
| ❌ | `cross-mark` | Wrong approach, mistake |
| 😢 | `cry` | Disappointment |
| 😡 | `angry` | Anger at situation |

### Facts/Education
| Emoji | Name | Use for |
|-------|------|---------|

| 🤔 | `thinking-face` | Question, reflection |
| ⏰ | `alarm-clock` | Time-related, urgency |
| 🤓 | `nerd-face` | Technical detail |
| 🧐 | `monocle` | Analyzing, looking closer |

## All Available Names

Run `getAvailableEmojis()` from `@remotion/animated-emoji` for the full list (~300 emojis).

Common ones: `smile`, `grin`, `joy`, `wink`, `heart-eyes`, `star-struck`, `partying-face`, `mind-blown`, `screaming`, `sleep`, `weary`, `thinking-face`, `money-face`, `sunglasses-face`, `robot`, `fire`, `sparkles`, `check-mark`, `cross-mark`, `light-bulb`, `muscle`, `thumbs-up`, `clap`, `wave`, `rocket`, `bell`, `100`, `red-heart`, `broken-heart`.
